이 스킬은 일반적인 "AI 슬롭" 미적 감각을 피하면서 독특하고 프로덕션 수준의 프론트엔드 인터페이스를 만드는 것을 안내합니다. 미적 세부사항과 창의적 선택에 탁월한 주의를 기울여 실제로 작동하는 코드를 구현합니다.

사용자가 프론트엔드 요구사항을 제공합니다: 컴포넌트, 페이지, 애플리케이션, 또는 인터페이스. 목적, 대상, 또는 기술적 제약에 대한 컨텍스트를 포함할 수 있습니다.

## 디자인 사고

코딩 전에 컨텍스트를 이해하고 BOLD한 미적 방향을 결정합니다:
- **목적**: 이 인터페이스는 어떤 문제를 해결하나요? 누가 사용하나요?
- **톤**: 극단을 선택하세요: 극도로 미니멀, 맥시멀리스트 카오스, 레트로-퓨처리스틱, 유기적/자연적, 럭셔리/정제된, 장난스러운/토이-같은, 에디토리얼/매거진, 브루탈리스트/로우, 아르데코/기하학적, 소프트/파스텔, 산업적/실용적 등. 선택지는 많습니다. 영감으로 활용하되 미적 방향에 충실한 디자인을 만드세요.
- **제약**: 기술적 요구사항 (프레임워크, 성능, 접근성).
- **차별화**: 이 디자인을 잊을 수 없게 만드는 것은 무엇인가요? 누군가 기억할 한 가지는 무엇인가요?

**중요**: 명확한 개념적 방향을 선택하고 정밀하게 실행합니다. 대담한 맥시멀리즘과 정제된 미니멀리즘 모두 효과적입니다 — 핵심은 강도가 아니라 의도입니다.

그런 다음 다음과 같은 작동하는 코드(HTML/CSS/JS, React, Vue 등)를 구현합니다:
- 프로덕션 수준이고 기능적
- 시각적으로 두드러지고 기억에 남는
- 명확한 미적 관점으로 일관된
- 모든 세부사항이 세심하게 다듬어진

## 프론트엔드 미적 가이드라인

다음에 집중합니다:
- **타이포그래피**: 아름답고 독특하며 흥미로운 폰트를 선택합니다. Arial이나 Inter 같은 일반적인 폰트를 피하고, 프론트엔드의 미적 감각을 높이는 독특한 선택을 합니다; 예상치 못하고 개성 있는 폰트 선택. 독특한 디스플레이 폰트와 정제된 본문 폰트를 페어링합니다.
- **색상 및 테마**: 일관된 미적 감각을 확립합니다. 일관성을 위해 CSS 변수를 사용합니다. 샤프한 강조색과 함께하는 지배적인 색상이 소심하고 균등하게 분배된 팔레트보다 효과적입니다.
- **모션**: 효과와 마이크로 인터랙션을 위해 애니메이션을 사용합니다. HTML에는 CSS 전용 솔루션을 우선시합니다. 가능하면 React에 Motion 라이브러리를 사용합니다. 고영향력 순간에 집중합니다: 단계별 reveal(animation-delay)이 있는 잘 조율된 페이지 로드 하나가 산발적인 마이크로 인터랙션보다 더 큰 즐거움을 줍니다. 스크롤 트리거링과 놀라움을 주는 hover 상태를 사용합니다.
- **공간 구성**: 예상치 못한 레이아웃. 비대칭. 겹침. 대각선 흐름. 그리드를 깨는 요소들. 넉넉한 네거티브 스페이스 또는 통제된 밀도.
- **배경 및 시각적 세부사항**: 단색 기본값 대신 분위기와 깊이를 만듭니다. 전체 미적 감각에 맞는 맥락적 효과와 질감을 추가합니다. 그래디언트 메시, 노이즈 질감, 기하학적 패턴, 레이어드 투명도, 극적인 그림자, 장식적 테두리, 커스텀 커서, 그레인 오버레이 같은 창의적 형태를 적용합니다.

과도하게 사용된 폰트 패밀리(Inter, Roboto, Arial, 시스템 폰트), 상투적인 색상 계획(특히 흰색 배경에 보라색 그래디언트), 예측 가능한 레이아웃과 컴포넌트 패턴, 컨텍스트에 특화된 개성이 없는 쿠키-커터 디자인 같은 일반적인 AI 생성 미적 감각을 절대 사용하지 마세요.

창의적으로 해석하고 해당 컨텍스트를 위해 진정으로 디자인된 것처럼 느껴지는 예상치 못한 선택을 합니다. 어떤 디자인도 같아서는 안 됩니다. 밝은 테마와 어두운 테마, 다른 폰트, 다른 미적 감각을 다양하게 사용합니다. 세대 간에 공통 선택(예: Space Grotesk)으로 수렴하지 마세요.

**중요**: 구현의 복잡도를 미적 비전에 맞춥니다. 맥시멀리스트 디자인은 광범위한 애니메이션과 효과가 있는 정교한 코드가 필요합니다. 미니멀리스트 또는 정제된 디자인은 절제, 정밀함, 그리고 간격, 타이포그래피, 미묘한 세부사항에 대한 세심한 주의가 필요합니다. 우아함은 비전을 잘 실행하는 데서 옵니다.

기억하세요: Claude는 특별한 창의적 작업을 할 수 있습니다. 창의적 사고를 하고 독특한 비전에 완전히 헌신할 때 진정으로 무엇을 만들 수 있는지 보여주세요.
