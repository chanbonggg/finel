# 콘텐츠 엔진

하나의 아이디어를 모든 플랫폼에 동일하게 게시하는 대신, 각 플랫폼에 최적화된 강력한 콘텐츠로 전환합니다.

## 활성화 시점

- X 게시물 또는 스레드 작성 시
- LinkedIn 게시물 또는 런치 업데이트 초안 작성 시
- 짧은 영상 또는 YouTube 설명 영상 스크립트 작성 시
- 기사, 팟캐스트, 데모, 또는 문서를 소셜 콘텐츠로 재활용 시
- 런치, 마일스톤, 또는 테마를 중심으로 간단한 콘텐츠 계획 수립 시

## 첫 번째 질문

다음을 명확히 합니다:
- 소스 자산: 무엇을 기반으로 조정할 것인지
- 청중: 빌더, 투자자, 고객, 운영자, 또는 일반 청중
- 플랫폼: X, LinkedIn, TikTok, YouTube, 뉴스레터, 또는 멀티 플랫폼
- 목표: 인지도, 전환, 채용, 권위 확립, 런치 지원, 또는 참여

## 핵심 규칙

1. 플랫폼에 맞게 조정합니다. 같은 카피를 교차 게시하지 않습니다.
2. 훅이 요약보다 중요합니다.
3. 모든 게시물은 하나의 명확한 아이디어를 담아야 합니다.
4. 구호보다 구체적인 내용을 사용합니다.
5. 행동 유도 문구는 작고 명확하게 유지합니다.

## 플랫폼별 가이드

### X
- 빠르게 시작
- 게시물 또는 스레드의 각 트윗당 하나의 아이디어
- 필요하지 않은 경우 본문에 링크 제외
- 해시태그 남용 금지

### LinkedIn
- 강력한 첫 번째 줄
- 짧은 단락
- 교훈, 결과, 시사점을 중심으로 더 명시적인 프레이밍

### TikTok / 짧은 영상
- 처음 3초가 주의를 끌어야 함
- 단순 내레이션이 아닌 비주얼을 중심으로 스크립트 작성
- 하나의 데모, 하나의 주장, 하나의 CTA

### YouTube
- 초반에 결과를 보여줌
- 챕터별로 구성
- 20-30초마다 비주얼 갱신

### 뉴스레터
- 관련 없는 항목 묶음이 아닌 하나의 명확한 관점 제공
- 섹션 제목을 훑어볼 수 있게 작성
- 도입 단락이 실제로 역할을 수행하게 함

## 재활용 흐름

기본 순서:
1. 앵커 자산: 기사, 영상, 데모, 메모, 또는 런치 문서
2. 3-7개의 원자적 아이디어 추출
3. 플랫폼 최적화 변형 작성
4. 출력물 간 중복 제거
5. CTA를 플랫폼 의도에 맞게 정렬

## 결과물

캠페인 요청 시 반환:
- 핵심 각도
- 플랫폼별 초안
- 선택적 게시 순서
- 선택적 CTA 변형
- 게시 전 필요한 누락 입력 사항

## 품질 기준

납품 전 확인:
- 각 초안이 해당 플랫폼에 자연스럽게 읽힘
- 훅이 강력하고 구체적
- 일반적인 과장 언어 없음
- 요청하지 않는 한 플랫폼 간 중복된 카피 없음
- CTA가 콘텐츠와 청중에 맞음
