# Theme Factory 스킬

이 스킬은 신중하게 선별된 색상 팔레트와 폰트 페어링이 포함된 전문적인 폰트 및 색상 테마 모음을 제공합니다. 테마가 선택되면 모든 아티팩트에 적용할 수 있습니다.

## 목적

프레젠테이션 슬라이드 덱이나 기타 아티팩트에 일관되고 전문적인 스타일을 적용하기 위해 이 스킬을 사용합니다. 각 테마는 다음을 포함합니다:
- 16진수 코드가 있는 일관된 색상 팔레트
- 헤더와 본문 텍스트를 위한 보완적인 폰트 페어링
- 다양한 컨텍스트와 독자에 적합한 독특한 시각적 정체성

## 사용 지침

슬라이드 덱이나 기타 아티팩트에 스타일을 적용하려면:

1. **테마 쇼케이스 표시**: 사용자가 모든 사용 가능한 테마를 시각적으로 볼 수 있도록 `theme-showcase.pdf` 파일을 표시합니다. 수정하지 말고 단순히 파일을 보여줍니다.
2. **선택 요청**: 덱에 어떤 테마를 적용할지 물어봅니다
3. **선택 대기**: 선택한 테마에 대한 명시적 확인을 받습니다
4. **테마 적용**: 테마가 선택되면, 선택된 테마의 색상과 폰트를 덱/아티팩트에 적용합니다

## 사용 가능한 테마

다음 10개의 테마를 `theme-showcase.pdf`에서 확인할 수 있습니다:

1. **Ocean Depths** - 전문적이고 차분한 해양 테마
2. **Sunset Boulevard** - 따뜻하고 생동감 있는 일몰 색상
3. **Forest Canopy** - 자연스럽고 안정적인 어스 톤
4. **Modern Minimalist** - 깔끔하고 현대적인 그레이스케일
5. **Golden Hour** - 풍부하고 따뜻한 가을 팔레트
6. **Arctic Frost** - 시원하고 선명한 겨울 영감 테마
7. **Desert Rose** - 부드럽고 세련된 먼지 낀 톤
8. **Tech Innovation** - 대담하고 현대적인 테크 미적 감각
9. **Botanical Garden** - 신선하고 유기적인 정원 색상
10. **Midnight Galaxy** - 극적이고 우주적인 깊은 톤

## 테마 세부사항

각 테마는 `themes/` 디렉토리에 다음을 포함한 완전한 사양과 함께 정의됩니다:
- 16진수 코드가 있는 일관된 색상 팔레트
- 헤더와 본문 텍스트를 위한 보완적인 폰트 페어링
- 다양한 컨텍스트와 독자에 적합한 독특한 시각적 정체성

## 적용 프로세스

선호하는 테마가 선택된 후:
1. `themes/` 디렉토리에서 해당 테마 파일을 읽습니다
2. 덱 전체에 지정된 색상과 폰트를 일관되게 적용합니다
3. 적절한 대비와 가독성을 보장합니다
4. 모든 슬라이드에서 테마의 시각적 정체성을 유지합니다

## 사용자 지정 테마 만들기
기존 테마 중 아티팩트에 맞는 것이 없는 경우, 사용자 지정 테마를 만듭니다. 제공된 입력을 기반으로 위의 것들과 유사한 새 테마를 생성합니다. 폰트/색상 조합이 나타내는 것을 설명하는 비슷한 이름을 테마에 부여합니다. 제공된 기본 설명을 사용해 적절한 색상/폰트를 선택합니다. 테마 생성 후, 검토 및 확인을 위해 표시합니다. 그런 다음 위에 설명된 대로 테마를 적용합니다.
